# note-pulp

## PuLPとは

PuLPは、ソルバーを使用して線形問題を解くライブラリ

- 対応ソルバー
  - GLPK
  - COIN-OR CLP/CBC (default)
  - CPLEX
  - GUROBI
  - MOSEK
  - XPRESS
  - CHOCO
  - MIPCL
  - SCIP
- デフォルト以外のソルバーを使用する場合は、別途インストールする必要がある
